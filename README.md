# laravel-blog-api
Backend api for nuxt-blog-site

### Front-End
Clone front-end at [nuxt-blog-site](https://github.com/reeshkeed/nuxt-blog-site)

### Install dependencies
```
$ composer install
```

### Start server

```
$ php artisan serve
```

### ENV file
Configure your .env file.

### Migrate database
```
$ php artisan migrate
```

### Database Seed
To generate dummy article data.

```
$ php artisan db:seed
```
